app.webcenter
==============
the Web UI for Dew management

Quick Start
---------------
##### Deploy and Run Dew in cluster.
##### Download DERBY Database and set DERBY_HOME in environment variables and set DERBY_HOME/bin in PATH
##### Create Database : cd app.webcenter and 
		
		./create-db.sh
##### Start WebCenter  

		./start-web.sh

##### Login

		Web link : IP:6077
		UserName : admin
		password : admin
